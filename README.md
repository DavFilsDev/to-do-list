# Enhanced To-Do List App

A feature-rich To-Do List application built with vanilla HTML, CSS, and JavaScript.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Delete tasks
- ✅ Filter tasks (All/Pending/Completed)
- ✅ Clear all completed tasks
- ✅ Task statistics counter
- ✅ Persistent storage with localStorage
- ✅ Responsive design
- ✅ Keyboard support (Enter to add)
- ✅ Visual notifications
- ✅ FontAwesome icons

## How to Use

1. Type your task in the input field
2. Click "Add" or press Enter to add the task
3. Click on a task to mark it as complete/incomplete
4. Click the × icon to delete a task
5. Use filter buttons to view specific tasks
6. Click "Clear Completed" to remove all completed tasks
